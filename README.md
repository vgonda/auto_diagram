# AutoDiagram
Auto generates a diagram of your Rails models every time you run a migration

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'auto_diagram'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install auto_diagram
```
